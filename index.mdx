## Quick Install

```bash
npm install -g @shadcn-component-manager/scm
```

## Get Started

<Steps>
  <Step title="Login">
    ```bash
    scm login
    ```
  </Step>
  <Step title="Search">
    ```bash
    scm search button
    ```
  </Step>
  <Step title="Preview">
    ```bash
    scm preview username/component-name
    ```
  </Step>
  <Step title="Install">
    ```bash
    scm add username/component-name
    ```
  </Step>
</Steps>

## Core Commands

<CodeGroup>
  <CodeGroupItem title="Authentication">
    ```bash
    scm login          # Authenticate with GitHub
    scm logout         # Clear authentication
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Discovery">
    ```bash
    scm search <query> # Search components
    scm preview <component> # Preview component details
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Installation">
    ```bash
    scm add <component> # Install component
    scm update [component] # Update components
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Creation">
    ```bash
    scm create <name>  # Create new component
    scm fork <component> # Fork existing component
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Publishing">
    ```bash
    scm publish        # Publish component
    ```
  </CodeGroupItem>
</CodeGroup>
