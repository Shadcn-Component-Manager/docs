# SCM Documentation

SCM (Shadcn Component Manager) is a CLI tool and registry ecosystem for discovering, installing, and managing UI components.

## Quick Install

```bash
npm install -g @shadcn-component-manager/scm
```

## Get Started

<Steps>
  <Step title="Login">
    ```bash
    scm login
    ```
  </Step>
  <Step title="Browse">
    ```bash
    scm browse
    ```
  </Step>
  <Step title="Search">
    ```bash
    scm search button
    ```
  </Step>
  <Step title="Install">
    ```bash
    scm add username/component-name
    ```
  </Step>
</Steps>

## Core Commands

<CodeGroup>
  <CodeGroupItem title="Authentication">
    ```bash
    scm login          # Authenticate with GitHub
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Discovery">
    ```bash
    scm search <query> # Search components
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Installation">
    ```bash
    scm add <component> # Install component
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Creation">
    ```bash
    scm create <name>  # Create new component
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Publishing">
    ```bash
    scm publish        # Publish component
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Updates">
    ```bash
    scm update         # Update components
    ```
  </CodeGroupItem>
</CodeGroup>
