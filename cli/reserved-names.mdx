# Reserved Component Names

Certain component names are reserved to avoid conflicts with official shadcn/ui components.

## What Are Reserved Names?

Reserved names are protected to ensure compatibility and prevent accidental overwrites of official components.

## How SCM Handles Reserved Names

<CardGroup cols={3}>
  <Card title="Installation">
    If you run `scm add button`, SCM redirects to the official component
  </Card>
  <Card title="Creation">
    If you try to create a reserved name, SCM blocks the operation
  </Card>
  <Card title="Publishing">
    Reserved names cannot be published to the registry by users
  </Card>
</CardGroup>

## Best Practices

<CardGroup cols={2}>
  <Card title="Naming">
    - Use descriptive, unique names for your components
    - Avoid using names that conflict with official components
    - Check the reserved names list before creating or publishing
  </Card>
  <Card title="Alternatives">
    - Use prefixes: `my-button`, `custom-button`
    - Use suffixes: `button-variant`, `button-enhanced`
    - Use descriptive names: `primary-button`, `animated-button`
  </Card>
</CardGroup>
