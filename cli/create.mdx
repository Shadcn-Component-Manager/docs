## Basic Usage

```bash
scm create <component-name>
```

## Options

```bash
# Component type (ui, hook, theme, block, page)
scm create my-component --type ui

# Component description
scm create my-component --description "A beautiful button component"

# Component author
scm create my-component --author "John Doe"

# Comma-separated categories
scm create my-component --categories "ui,button,interactive"

# Custom output path
scm create my-component --path custom/path

# Skip confirmation prompts
scm create my-component --yes

# Overwrite existing directory
scm create my-component --force

# Don't generate template files
scm create my-component --skip-template
```

## Example

```bash
scm create my-awesome-button
```

## What Happens

<Steps>
  <Step title="Validate Name">
    Checks if component name is reserved
  </Step>
  <Step title="Create Directory">
    Creates `components/<name>/` directory
  </Step>
  <Step title="Generate Files">
    Creates `registry.json` and component file
  </Step>
  <Step title="Verify Structure">
    Ensures proper file structure
  </Step>
</Steps>

## Generated Structure

```
components/my-awesome-button/
├── registry.json              # Component metadata
└── my-awesome-button.tsx      # Main component file
```

## Generated Files

### registry.json

```json
{
  "name": "my-awesome-button",
  "type": "registry:ui",
  "files": [
    {
      "path": "my-awesome-button.tsx",
      "type": "registry:ui"
    }
  ]
}
```

### Component File

```tsx
import * as React from "react"
import { cn } from "@/lib/utils"

export interface MyAwesomeButtonProps extends React.HTMLAttributes<HTMLDivElement> {
  // Add your component props here
}

const MyAwesomeButton = React.forwardRef<HTMLDivElement, MyAwesomeButtonProps>(
  ({ className, ...props }, ref) => {
    return (
      <div
        className={cn("", className)}
        ref={ref}
        {...props}
      >
        {/* Your component content */}
        MyAwesomeButton
      </div>
    )
  }
)
MyAwesomeButton.displayName = "MyAwesomeButton"

export { MyAwesomeButton }
```

## Reserved Names

<Callout>
  Certain names are reserved to avoid conflicts with official shadcn/ui components.
</Callout>

**Reserved Names:**
- button, card, dialog, input, form, label, select, textarea
- checkbox, radio, switch, slider, utils, theme
- All shadcn/ui component names
