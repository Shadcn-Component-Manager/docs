## Environment Variables

SCM respects these environment variables:

<CodeGroup>
  <CodeGroupItem title="GitHub Token">
    ```bash
    GITHUB_TOKEN=your_github_token
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Registry URL">
    ```bash
    SCM_REGISTRY_URL=https://raw.githubusercontent.com/Shadcn-Component-Manager/registry/main/components
    ```
  </CodeGroupItem>
  <CodeGroupItem title="Debug Mode">
    ```bash
    SCM_DEBUG=true
    ```
  </CodeGroupItem>
</CodeGroup>

## Authentication Setup

<Steps>
  <Step title="Login">
    ```bash
    scm login
    ```
  </Step>
  <Step title="Verify">
    Try an authenticated command like `scm publish`
  </Step>
  <Step title="Start Creating">
    ```bash
    scm create my-component
    ```
  </Step>
</Steps>
