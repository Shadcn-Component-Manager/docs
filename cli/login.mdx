---
title: 'Login Command'
description: 'Authenticate with SCM using GitHub OAuth device flow.'
---

## Basic Usage

```bash
scm login
```

## Options

```bash
# Force re-authentication
scm login --force

# Use existing token instead of OAuth flow
scm login --token your-github-token

# Check current authentication status
scm login --check

# Show detailed auth info
scm login --verbose
```

## Authentication Flow

<Steps>
  <Step title="Request Authorization">
    SCM generates device authorization code
  </Step>
  <Step title="Browser Authorization">
    Opens browser to GitHub authorization page
  </Step>
  <Step title="Token Exchange">
    Exchanges authorization code for access token
  </Step>
  <Step title="Store Token">
    Securely stores token for future use
  </Step>
</Steps>
