# Search Components

Search for components in the registry using keywords and filters.

## Basic Search

```bash
# Search by name
scm search button

# Search by description
scm search "dark mode"
```

## Search Filters

```bash
# Limit results
scm search button --limit 5

# Force refresh cache
scm search button --force
```

## Examples

```bash
# Find button components
scm search button

# Find dark mode components
scm search "dark mode"

# Find UI components with limit
scm search ui --limit 10
```
