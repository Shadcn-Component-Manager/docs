## Basic Search

```bash
# Search by name
scm search button

# Search by description
scm search "dark mode"
```

## Options

```bash
# Force refresh cache
scm search button --force

# Filter by category
scm search button --category ui

# Limit results
scm search button --limit 10

# Filter by author
scm search button --author username

# Sort by field (name, author, date, popularity)
scm search button --sort name

# Output format (table, json, csv)
scm search button --output table

# Filter by component type
scm search button --type ui

# Show detailed information
scm search button --verbose

# Minimal output
scm search button --quiet
```

## Examples

```bash
# Find button components
scm search button

# Find dark mode components
scm search "dark mode"

# Find UI components with limit
scm search ui --limit 10

# Search by author
scm search --author johndoe

# Filter by category
scm search button --category ui

# JSON output format
scm search button --output json

# Force refresh cache
scm search button --force

# Sort by popularity
scm search button --sort popularity
```
