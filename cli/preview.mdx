# Preview Command

Preview component details, documentation, and metadata before installing.

## Basic Usage

```bash
scm preview user/component
```

## Options

```bash
# Preview specific version
scm preview user/component --version 1.0.0
```

## Preview Information

<CardGroup cols={2}>
  <Card title="Component Metadata">

## Version Preview

```bash
# Preview latest version
scm preview user/component

# Preview specific version
scm preview user/component --version 1.0.0
```
