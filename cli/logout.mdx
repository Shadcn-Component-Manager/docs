# Logout Command

Clear stored GitHub credentials and authentication data.

## Basic Usage

```bash
scm logout
```

## Options

```bash
# Clear all cached data (not just token)
scm logout --all

# Clear cache only
scm logout --cache

# Force logout without confirmation
scm logout --force
```

## What Logout Does

<Steps>
  <Step title="Remove Token">
    Remove stored GitHub token from config file
  </Step>
  <Step title="Clear Cache">
    Clear cached component data
  </Step>
  <Step title="Confirm">
    Confirm the logout with success message
  </Step>
</Steps>

## Files Removed

The logout command removes:

- **Config File**: `~/.scm/config.json`
- **Cache Directory**: `~/.scm/cache/`

## Examples

```bash
# Basic logout
scm logout

# Clear all cached data
scm logout --all

# Clear cache only
scm logout --cache

# Force logout without confirmation
scm logout --force
```

## Verification

After logout, try an authenticated command:

```bash
scm publish
# Output: ‚ùå Authentication required. Please run 'scm login' first
```

## Re-authentication

After logging out, authenticate again:

```bash
scm login
```
