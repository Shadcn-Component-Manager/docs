# Add Components

Install components from the registry into your project.

## Basic Usage

```bash
# Install latest version
scm add username/component-name

# Install specific version
scm add username/component-name@1.0.0

# Force refresh cache
scm add username/component-name --force
```

## Options

```bash
# Force refresh cache
scm add username/component-name --force

# Show what would be installed without installing
scm add username/component-name --dry-run

# Custom installation path
scm add username/component-name --path custom/path

# Skip installing dependencies
scm add username/component-name --skip-deps

# Show detailed installation info
scm add username/component-name --verbose

# Skip confirmation prompts
scm add username/component-name --yes
```

## Examples

```bash
# Install a button component
scm add johndoe/awesome-button

# Install with specific version
scm add johndoe/awesome-button@2.1.0

# Force reinstall
scm add johndoe/awesome-button --force

# Dry run to see what would be installed
scm add johndoe/awesome-button --dry-run

# Skip dependency installation
scm add johndoe/awesome-button --skip-deps

# Install with verbose output
scm add johndoe/awesome-button --verbose
```

## Reserved Component Names

If you try to install a reserved component name (like `button`, `card`, etc.), SCM automatically redirects to the official shadcn/ui component:

```bash
scm add button
# Redirects to: npx shadcn@latest add button
```

## File Structure

Components are installed to your project's component directory:

```
components/
├── ui/
│   └── component-name.tsx    # Main component
├── hooks/
│   └── use-component.ts      # Custom hooks
└── lib/
    └── component-utils.ts    # Utilities
```

## Requirements

- `components.json` file (from shadcn/ui setup)
- Valid component name in registry
- Internet connection for downloads

## Troubleshooting

**Component not found:**
```bash
# Search for similar components
scm search component-name

# Check if component exists
scm preview username/component-name
```

**Installation fails:**
```bash
# Clear cache and retry
scm add component --force

# Check dependencies manually
npm install
```
