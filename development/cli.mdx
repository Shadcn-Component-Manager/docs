## Prerequisites

- Node.js 18+
- Git with GitHub account
- pnpm (recommended)

## Setup

```bash
git clone https://github.com/Shadcn-Component-Manager/scm.git
cd scm
pnpm install
```

## Project Structure

```
scm/
├── src/
│   ├── cli.ts              # Main entry point
│   ├── commands/           # Command implementations
│   └── lib/               # Core libraries
├── dist/                  # Compiled output
└── scripts/              # Build scripts
```

## Adding Commands

1. Create file in `src/commands/`
2. Follow existing pattern
3. Register in main CLI file
4. Test locally

```typescript
import { Command } from 'commander';

export const newCommand = new Command('new')
  .description('New command description')
  .action(async () => {
    // Implementation
  });
```

## Build Process

- TypeScript compilation
- esbuild bundling
- Binary generation with pkg
- Development watch mode

## Publishing

1. Update version in package.json
2. `pnpm build`
3. `pnpm start --help` (test)
4. `pnpm publish`
