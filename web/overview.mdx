# Web Interface

The SCM web interface provides a visual way to browse, search, and explore components from the registry.

## Features

<CardGroup cols={2}>
  <Card title="Component Grid" icon="grid">
    Browse all components in a responsive grid layout
  </Card>
  <Card title="Component Details" icon="eye">
    View detailed component information and source code
  </Card>
  <Card title="File Viewer" icon="code">
    Browse component files with syntax highlighting
  </Card>
  <Card title="Version Selection" icon="git-branch">
    Switch between different component versions
  </Card>
</CardGroup>
