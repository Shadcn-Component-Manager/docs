# Browsing Components

Browse the component registry to discover components for your projects.

## Components Page

Visit `/components` to see all available components in a responsive grid layout.

### Component Cards

Each component displays:

<CardGroup cols={2}>
  <Card title="Name & Author" icon="user">
    `author/component-name` format
  </Card>
  <Card title="Version" icon="tag">
    Latest version badge
  </Card>
  <Card title="Description" icon="file-text">
    Brief component description
  </Card>
  <Card title="Categories" icon="tags">
    Up to 3 category tags
  </Card>
</CardGroup>

### Grid Layout

<CardGroup cols={2}>
  <Card title="Desktop" icon="monitor">
    3 columns
  </Card>
  <Card title="Tablet" icon="tablet">
    2 columns
  </Card>
  <Card title="Mobile" icon="smartphone">
    1 column
  </Card>
</CardGroup>

<CardGroup cols={2}>

  </Card>
</CardGroup>
