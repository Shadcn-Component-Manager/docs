# Component Details

View detailed information about components including source code and documentation.

## Component Page

Visit `/components/[author]/[name]` to view a specific component.

### Header Section

  </Card>
  <Card title="Component Name" icon="tag">
    Displays component name and title
  </Card>
  <Card title="Description" icon="file-text">
    Full component description
  </Card>
  <Card title="Version Selector" icon="git-branch">
    Dropdown to switch between versions
  </Card>
</CardGroup>

### Main Content
