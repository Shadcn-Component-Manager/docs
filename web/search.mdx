# Search Components

Find components using the search functionality.

## Search Bar

The search bar is available at the top of every page.

### Search Types

<CardGroup cols={2}>
  <Card title="Component Names" icon="tag">
    Search by exact or partial component names
  </Card>
  <Card title="Author Names" icon="user">
    Find components by specific authors
  </Card>
  <Card title="Descriptions" icon="file-text">
    Search within component descriptions
  </Card>
  <Card title="Categories" icon="tags">
    Filter by component categories
  </Card>
</CardGroup>
